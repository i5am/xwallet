🎯 推荐方案 (按优先级)

┌─ 方案 1: GitHub Actions ────────────────────────────────────┐
│ 💰 成本: 免费 (公开仓库)                                        │
│ 🔧 难度: 中等                                                   │
│ ⏱️  时间: 10-15 分钟/次构建                                     │
│ ✅ 优势: 最适合 Capacitor, 完全免费                             │
│                                                                │
│ 📁 配置文件: .github/workflows/ios-build.yml (已创建)          │
│                                                                │
│ 📝 需要的步骤:                                                  │
│   1. 在 Mac 上生成证书 (或租云 Mac 1小时 ~$1)                  │
│   2. 添加 4 个 GitHub Secrets                                  │
│   3. 推送代码或创建 tag                                         │
│   4. 下载 IPA 文件                                             │
└────────────────────────────────────────────────────────────┘

┌─ 方案 2: Ionic Appflow ─────────────────────────────────────┐
│ 💰 成本: 免费层 (500分钟/月) 或 $49/月                         │
│ 🔧 难度: 简单                                                   │
│ ⏱️  时间: 5-10 分钟/次构建                                      │
│ ✅ 优势: 专为 Capacitor 设计, 零配置                           │
│                                                                │
│ 🚀 快速开始:                                                    │
│   npm install -g @ionic/cli                                   │
│   ionic login                                                 │
│   ionic link                                                  │
│   ionic package build ios                                     │
└────────────────────────────────────────────────────────────┘

┌─ 方案 3: 本地 Mac 构建 ─────────────────────────────────────┐
│ 💰 成本: 需要 Mac 硬件                                          │
│ 🔧 难度: 简单                                                   │
│ ⏱️  时间: 5 分钟/次构建                                         │
│ ✅ 优势: 最可靠, 最快                                           │
│                                                                │
│ 🚀 快速开始:                                                    │
│   npx cap sync ios                                            │
│   npx cap open ios                                            │
│   (在 Xcode 中: Product → Archive)                            │
└────────────────────────────────────────────────────────────┘

┌─ 方案 4: 云 Mac 服务 ───────────────────────────────────────┐
│ 💰 成本: $1/小时 或 $30/月                                      │
│ 🔧 难度: 简单                                                   │
│ ⏱️  时间: 5-10 分钟/次构建                                      │
│ ✅ 优势: 不需要拥有 Mac                                         │
│                                                                │
│ 🌐 服务商:                                                      │
│   - MacinCloud: https://www.macincloud.com                    │
│   - AWS EC2 Mac: https://aws.amazon.com/ec2/instance-types/mac│
│   - MacStadium: https://www.macstadium.com                    │
└────────────────────────────────────────────────────────────┘

🚀 最快上手方案: GitHub Actions

1️⃣  准备证书 (一次性, ~30分钟)
   - 借用朋友的 Mac, 或
   - 租云 Mac 1小时 ($1-5)

2️⃣  配置 GitHub (5分钟)
   - 添加 4 个 Secrets
   - 推送代码

3️⃣  等待构建 (10-15分钟)
   - 自动构建
   - 下载 IPA

总成本: ~$1-5 (一次性)
未来构建: 完全免费!
